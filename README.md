# APP_WEB_SEMANA5

# Galería de Imágenes Interactiva - Semana 5

## Descripción
Aplicación web dinámica desarrollada con HTML, CSS y JavaScript que permite:
- Agregar imágenes mediante URL.
- Seleccionar una imagen (resaltada con borde verde).
- Eliminar la imagen seleccionada.

## Funcionalidades
- Uso de `createElement`, `addEventListener`, y manipulación del DOM.
- Soporte para tecla `Enter` al ingresar URL.
- Validación básica de URL e imágenes.
- Diseño responsivo con CSS Grid.

## Cómo usar
1. Abre `index.html` en el navegador (compatible con Windows 11).
2. Ingresa una URL de imagen (ej: https://picsum.photos/200).
3. Haz clic en "Agregar Imagen" o presiona Enter.
4. Haz clic en cualquier imagen para seleccionarla.
5. Usa "Eliminar Imagen Seleccionada" para borrarla.

> Nota: Las imágenes deben ser accesibles públicamente.
